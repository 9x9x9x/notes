# UPSC CSE Static Knowledge Base

Welcome to your central static repository. This engine processes raw Markdown notes straight into a scannable, search-indexed document database layout.

### Run

    python3 -m http.server 3000

### Core Instructions for Adding Notes
1. Create a new standard markdown file in this folder matching your link target (e.g., `GS2_Constitutional_Structure.md`).
2. Open `_sidebar.md` and add a bullet link under the appropriate header.
3. Refresh your browser to view your live, formatted updates instantly.
